RSA Keypairs

RSA is an asymmetric encryption algorithm that uses a pair of mathematically related keys:

  • Public Key: Can be freely shared. Used to encrypt data and verify signatures.
  • Private Key: Must be kept secret. Used to decrypt data and create signatures.

Security Specifications

  • Key Size: 2048 bits
  • Public Exponent: 65537
  • Format: PEM encoding
  • Security Level: 112-bit equivalent

Key Uses

  • RSA-OAEP: Encrypts symmetric keys (key wrapping)
  • RSA-PSS: Creates digital signatures for authentication
  • Combined: Enables hybrid cryptography with AES-GCM

Security Best Practices

  • Backup your private key securely
  • Rotate keys periodically
